# CHANGELOG

## [0.1.1] - 2025-12-30
- 规划功能：内存限制功能，支持设置最大内存上限
- 规划功能：LRU淘汰策略，超内存时自动淘汰最久未使用key
- 规划功能：内存使用量统计方法TotalSize()

## [0.1.0] - 2025-11-17
### Added
- 初始化项目结构，搭建核心目录架构
- 实现基础缓存功能：线程安全的Set/Get/Delete操作
- 基于sync.RWMutex保证高并发访问安全
- 新增缓存条目结构体，记录key-value及基础元信息
- 补充阶段1基础使用示例与单元测试

### 版本说明
- 遵循语义化版本（Semantic Versioning）：主版本.次版本.修订版本
- 主版本：不兼容的API变更
- 次版本：向后兼容的功能新增
- 修订版本：向后兼容的问题修复
